# S3 to Spark to Kafka

This project reads data from an S3 bucket, parses it using Apache Spark, and pushes the processed data into a Kafka topic.

## Overview

- **Source**: Amazon S3 bucket
- **Processing**: Apache Spark
- **Destination**: Apache Kafka topic

## Features

- Read data from S3 buckets
- Parse and transform data using Spark
- Stream processed data to Kafka topics
- Scalable data pipeline architecture
